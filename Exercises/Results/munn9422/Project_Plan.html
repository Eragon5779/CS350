Feature Set

Wall Feature

As​ ​a​ ​user,​ ​I​ ​want​ ​to​ ​upload​ ​multimedia​ ​content​ ​so​ ​that​ ​users​ ​within​ ​my friend network can comment, view and read my content.
  • A​ ​user​ ​is​ ​able​ ​to​ ​post​ ​status​ ​updates​ ​consisting​ ​of​ ​text,​ ​images,​ ​videos,​ ​or​ ​links​ ​to​ ​their wall.
  • A​ ​user​ ​is​ ​able​ ​to​ ​comment​ ​on​ ​their​ ​status​ ​update​ ​which​ ​can​ ​also​ ​consist​ ​of​ ​multimedia content.
  • A​ ​user​ ​is​ ​able​ ​to​ ​share​ ​other​ ​user’s​ ​status​ ​updates​ ​to​ ​their​ ​own​ ​wall.
  • The​ ​wall​ ​shows​ ​a​ ​chronological​ ​feed​ ​of​ ​past​ ​status​ ​updates​ ​which​ ​can​ ​be​ ​commented​ ​on or​ ​viewed​ ​just​ ​like​ ​within​ ​the​ ​wall

Newsfeed Feature

As​ ​a​ ​user,​ ​I​ ​want​ ​to​ ​browse​ ​the​ ​recent​ ​content​ ​shared​ ​by​ ​my​ ​friend​ ​network​ ​so​ ​that​ ​I​ ​can comment,​ ​view​ ​and​ ​read​ ​what​ ​was​ ​shared​ ​by​ ​my​ ​friends.
  • A​ ​user​ ​is​ ​able​ ​to​ ​browse​ ​content​ ​shared​ ​or​ ​updated​ ​by​ ​people​ ​within​ ​their​ ​friend​ ​network.
  • Along​ ​with​ ​the​ ​content​ ​within​ ​each​ ​update​ ​that​ ​is​ ​posted,​ ​the​ ​user​ ​who​ ​posted​ ​the​ ​content and​ ​a​ ​timestamp​ ​of​ ​the​ ​post​ ​are​ ​also​ ​shared .
  • Users​ ​are​ ​able​ ​to​ ​comment​ ​on​ ​posts​ ​they​ ​see​ ​while​ ​browsing​ ​their​ ​newsfeed.
  • Users​ ​are​ ​able​ ​to​ ​view​ ​videos​ ​within​ ​the​ ​windows​ ​that​ ​they​ ​post.

Friend Feature

As​ ​a​ ​user,​ ​I​ ​want​ ​to​ ​connect​ ​or​ ​disconnect​ ​with​ ​another​ ​person​ ​so​ ​that​ ​I​ ​can​ ​control​ ​who​ ​is​ ​within my​ ​network.
  • A​ ​user​ ​is​ ​able​ ​to​ ​search​ ​for​ ​friends​ ​by​ ​name.
  • A​ ​user​ ​is​ ​able​ ​‘become​ ​friends’​ ​with​ ​another​ ​user.​ ​Users​ ​send​ ​requests​ ​to​ ​other​ ​users, and​ ​requests​ ​can​ ​be​ ​accepted​ ​or​ ​denied.
  • A​ ​user​ ​is​ ​able​ ​to​ ​unfriend​ ​someone​ ​after​ ​they​ ​have​ ​friended​ ​them.
  • After​ ​a​ ​user​ ​is​ ​friends​ ​with​ ​another​ ​user,​ ​then​ ​they​ ​can​ ​see​ ​each​ ​other’s​ ​shared​ ​content.

Private Message Feature

As​ ​a​ ​user,​ ​I​ ​want​ ​to​ ​be​ ​able​ ​to​ ​send​ ​messages​ ​to​ ​other​ ​users​ ​so​ ​that​ ​I​ ​can​ ​communicate​ ​without other​ ​people​ ​viewing​ ​the​ ​conversation.
  • A​ ​user​ ​is​ ​able​ ​to​ ​search​ ​for,​ ​another​ ​user​ ​and​ ​open​ ​a​ ​private​ ​messaging​ ​window.
  • A​ ​user​ ​is​ ​directly​ ​notified​ ​by​ ​email​ ​when​ ​another​ ​user​ ​initiates​ ​a​ ​conversation​ ​with​ ​them for​ ​the​ ​first​ ​time.
  • A​ ​user​ ​does​ ​not​ ​have​ ​to​ ​be​ ​friends​ ​in​ ​order​ ​to​ ​message​ ​someone.
  • A​ ​user​ ​can​ ​block​ ​another​ ​user​ ​from​ ​messaging​ ​them.
